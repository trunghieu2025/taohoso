========================================
  HƯỚNG DẪN CÀI ĐẶT PHẦN MỀM TẠO HỒ SƠ
========================================

YÊU CẦU HỆ THỐNG:
  - Windows 10 hoặc 11 (64-bit)
  - RAM: 4GB trở lên
  - Ổ cứng: 500MB trống

----------------------------------------

BƯỚC 1: Mở file cài đặt
  Double-click vào file "TaoHoSo-Setup-1.0.0.exe"

----------------------------------------

BƯỚC 2: Xử lý cảnh báo Windows (chỉ hiện 1 lần)

  Do phần mềm mới phát hành, Windows có thể hiện
  bảng cảnh báo màu xanh. Đây là bình thường.

  2a. Bấm vào dòng chữ "More info" (Thông tin thêm)
  2b. Bấm nút "Run anyway" (Vẫn chạy)

  (*) Cảnh báo này chỉ hiện 1 lần duy nhất khi cài đặt.

----------------------------------------

BƯỚC 3: Chọn ngôn ngữ
  Chọn "Tiếng Việt" hoặc "English" -> bấm OK

----------------------------------------

BƯỚC 4: Cài đặt
  1. Bấm "Tiếp theo" (Next)
  2. Chọn thư mục cài đặt (hoặc để mặc định)
  3. Bấm "Cài đặt" (Install)
  4. Đợi quá trình cài đặt hoàn tất
  5. Bấm "Hoàn tất" (Finish)

----------------------------------------

BƯỚC 5: Sử dụng
  Mở phần mềm bằng 1 trong 2 cách:
  - Icon "Tao Ho So" trên Desktop
  - Tìm "Tao Ho So" trong Start Menu

----------------------------------------

GỠ CÀI ĐẶT (nếu cần):
  1. Mở Settings (Cài đặt Windows)
  2. Vào Apps (Ứng dụng)
  3. Tìm "Tao Ho So"
  4. Bấm Uninstall (Gỡ cài đặt)

  người quản lý.
