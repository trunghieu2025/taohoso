# Bắt đầu phiên làm việc

## Bước 1: Pull code mới nhất

// turbo

```
git pull origin hieu1
```

## Bước 2: Kiểm tra trạng thái

// turbo

```
git log --oneline -5
```

## Bước 3: Chạy dev server

```
npm run dev
```

> [!IMPORTANT]
> Luôn pull trước khi bắt đầu code để tránh conflict với phiên trước.
